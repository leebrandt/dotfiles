# InkyPinky
InkyPinky brings a soft, light-pink glow to Omarchy — gentle on the eyes and full of charm. 
This delicate theme adds warmth and style, blending simplicity with a touch of elegance. 
Smooth, bright, and uplifting, InkyPinky makes your dashboard feel fresh, friendly, and full of life

# Installation Theme

To install this theme, simply use the omarchy-theme-install command:

```bash
omarchy-theme-install https://github.com/HANCORE-linux/omarchy-inkypinky-theme.git
```
