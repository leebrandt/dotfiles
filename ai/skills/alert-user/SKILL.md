---
name: alert-user
description: Use this when you need to alert the user of a decision you need from them, or if a failure has occurred.
---

IF you need to alert the user to:

- A decision that needs to be made before an agent can continue:
  - prefix with `⚠️ DECISION NEEDED:` State the decision needed in a short, direct way.
- A failure occurred while an agent was working on a spec:
  - prefix with `🚨 FAILURE:`: Describe the failure and wait for user input.
